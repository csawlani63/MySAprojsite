My name is Chirag Sawlani and the person you see above this text is me. I built this website for my something awesome project using HTML, CSS and JS, alongside a simple windows malware using C. The icons on the top-left of your screen take you to the home page, metadata webpage, malware webpage, GitHub page for the website and my Instagram profile respectively. This sidebar is on each webpage for ease of navigation.
The point of this website is to teach you about your leaking metadata and malware with a webpage and showcase for each. The purpose is to teach users information regarding their security that they may not be aware of and ways to reinforce that security to minimize browser leaks and data breaches from malware. Therefore, the malware and metadata sniffer are not too complex and relatively simple since my purpose was to teach so I went for a more informative approach with a show and tell kind of system.
Enjoy the website!
